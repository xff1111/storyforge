读图顺序

先看中间 1 到 8：它是正文生成主线。再看上方四层记忆如何在 ② 读入、⑥ 写回，最后看下方护栏如何贴着各阶段工作。

关键判断

一致性不是一个孤立功能，而是“读上下文 → 生成 → 写回记忆 → 校验 → 失效传播 → 再读”的闭环。

软硬边界

AI 负责生成和抽取，代码负责过滤、投影、校验、溯源和传播；作者确认把候选事实推成 canon。
